# Contributing

Thanks for wanting to improve RepoSpark.

## Good first contributions

- Add new checklist items for different project types.
- Improve the README preview copy.
- Add export-to-Markdown behavior.
- Add local save history.
- Improve mobile spacing or accessibility.

## Local setup

```bash
npm install
npm run dev
```

Before opening a pull request, run:

```bash
npm run build
```

## Pull request style

Keep changes focused. Include a short description of what changed, why it helps repo owners, and how you tested it.
